El Simple File Manager és un gestor de fitxers i carpetes molt ràpid i professional per a dispositius Android. Utilitzeu el Simple File Manager per comprimir, transferir i convertir fàcilment fitxers multimèdia amb uns quants clics. Té totes les funcions principals de gestió de fitxers i carpetes, inclosa la personalització de la carpeta d'inici i la selecció de carpetes preferides per accedir-hi ràpidament.

El gestor de fitxers ofereix tot el paquet de funcions de gestió de fitxers, com ara cercar, navegar, copiar i enganxar, tallar, suprimir, canviar el nom, descomprimir, transferir, descarregar, organitzar, etc. Afegiu, elimineu o editeu fitxers, carpetes i aplicacions segons les vostres preferències personals.

Amb aquest organitzador senzill de dades, podeu organitzar i ordenar el mòbil segons diverses mètriques i alternar entre ascendent i descendent o utilitzar una ordenació específica de carpeta. Per obtenir ràpidament un camí d'arxiu o carpeta, podeu seleccionar-lo fàcilment prement i copiant-lo al porta-retalls.

El Simple File Manager facilita l'organització dels fitxers, carpetes i aplicacions mòbils per estalviar-vos temps i energia. Amb només uns quants clics, també podeu comprovar les propietats dels fitxers o carpetes, que mostren diversos camps com la mida, la data de l'última modificació o els valors EXIF com la data de creació, el model de càmera a les fotos, etc.

Aquest organitzador de fitxers és absolutament segur i conté múltiples funcions potents relacionades amb la seguretat, com ara protegir amb contrasenya els elements ocults, suprimir o obrir tota l'aplicació. Podeu triar entre utilitzar un patró, un pin o un bloqueig biomètric per mantenir les vostres dades privades. El permís d'empremta digital és necessari per bloquejar la visibilitat dels elements ocults, per suprimir fitxers o per a tota l'aplicació. El Simple File Manager funciona sense accés a Internet, garantint encara més la vostra privadesa definitiva.

El gestor de fitxers també pot netejar espai i estalviar l'emmagatzematge intern comprimint fitxers i carpetes. Aquest organitzador modern de fitxers multimèdia admet la navegació ràpida de fitxers arrel, targetes SD i dispositius USB. El gestor de fitxers també reconeix diversos formats de fitxer, com ara música, vídeos, imatges i documents.

Utilitzeu el Simple File Manager per crear dreceres d'escriptori útils per accedir ràpidament als elements preferits. Conté un editor lleuger de fitxers que podeu utilitzar per imprimir documents, editar-los o llegir-los fàcilment amb gestos de zoom, sempre que sigui necessari.

Tot i que es diu Simple File Manager, us ajudarà a gestionar i personalitzar els vostres fitxers, carpetes i aplicacions amb només uns quants clics. Podeu veure fàcilment els vostres fitxers recents i també fer una anàlisi d'emmagatzematge.

Podeu utilitzar l'anàlisi d'emmagatzematge integrat per tenir una visió general ràpida sobre quins fitxers ocupen més espai i netejar-lo. Pot funcionar com a netejador d'emmagatzematge que us ajudarà a buidar una mica d'espai en el dispositiu.

Ve amb el Material Design i un tema fosc per defecte, proporciona una gran experiència d'usuari per a un ús fàcil. La manca d'accés a Internet us ofereix més privadesa, seguretat i estabilitat que altres aplicacions.

No conté anuncis ni permisos innecessaris. És totalment de codi obert, proporciona colors personalitzables.
